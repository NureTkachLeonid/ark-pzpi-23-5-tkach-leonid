1. МЕТА
Вивчити стандарти кодування (Code Convention) для підвищення читабельності та підтримки коду в проєкті "Smart Plant Care".

2. ХІД РОБОТИ
Для забезпечення якості коду в своєму проєкті я обрав наступні правила:

1. Стиль іменування (Naming Convention): 
Я буду використовувати CamelCase для імен змінних (наприклад, soilMoisture, lightLevel) та методів (наприклад, getWaterLevel, turnOnPump). Для назв класів буде використовуватись PascalCase (наприклад, PlantSensor, PumpController). Це дозволить легко відрізняти типи даних від змінних та функцій.

2. Форматування (Formatting):
Для відступів я буду використовувати 4 пробіли, оскільки це є стандартом для багатьох C-подібних мов. Фігурні дужки будуть розміщуватися на новому рядку (стиль Allman). Такий стиль робить блоки коду візуально чіткішими, що важливо при налагодженні логіки роботи контролерів.

3. Коментарі (Comments):
Код має бути самодокументованим, тобто назви змінних та методів мають чітко пояснювати їх призначення. Проте, для документування публічних методів API та складних алгоритмів обробки даних з датчиків я буду використовувати формат Javadoc/Doxygen.

4. Константи:
Для уникнення "магічних чисел" у коді я буду використовувати іменовані константи, записані у форматі UPPER_CASE. Наприклад, замість того, щоб писати число 100 у перевірках, я створю константу MAX_WATER_LEVEL = 100. Це полегшить зміну налаштувань системи в майбутньому.

3. ПРИКЛАДИ КОДУ

Нижче наведено приклад класу на мові C++, що демонструє обрані правила:

class SensorData {
public:
    // Константа для максимального рівня вологості
    static const int MAX_MOISTURE_LEVEL = 100;

    // Конструктор класу
    SensorData(int initialMoisture) {
        currentMoisture = initialMoisture;
    }

    // Метод для отримання рівня вологості
    int getMoistureLevel() const {
        return currentMoisture;
    }

    // Метод для перевірки, чи потрібен полив
    bool needsWatering(int threshold) {
        if (currentMoisture < threshold) {
            return true;
        }
        return false;
    }

private:
    int currentMoisture; // Поточне значення вологості
};
